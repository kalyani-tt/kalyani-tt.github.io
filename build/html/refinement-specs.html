Remark 🗣️ Refinement and Program Specifications

Definition ✍️ Refinement

We say that something refines something else if it is "more specific" is some sense, e.g:

  1. In a state transition system, one transition relation T refines another T' if ∀s t, T s t → T' s t
  2. A strategy V refines another V' if ∀s, s ∈ V → s ∈ V'

We can state that a program meets a specification using refinement. All we need is a specification and a way of interpreting programs as specifications. For instance a specification might be a state transition system describing that valid transitions the program can make, and the interpretation of the program is the transition system describing how the program actually executes. We then prove that the program refines the spec.
