Idea 💡 Types as Predicates

Instead of defining types as a built-in notion, limited to e.g struct, enum, int, etc, types are predicates. So we can take any sort of property we want a value to have, and turn that into a type. We can then interpret the types as a logic programming language and execute the types in that way to get a typechecker.

Here's a sketch of what it could look like (where self is the value being typechecked) 

type positive :=
    self = 0 \/
    exists n,
        positive(n) /\
        self = 1 + n

let n: positive = 1 + 1 + 1 + 0

And then the compiler executes positive(1 + 1 + 1 + 0) as a logic program to typecheck n.
